# 🎵 MusicBot - پخش موسیقی در ماینکرفت

پلاگین اسپیگات برای پخش موسیقی از طریق **Simple Voice Chat**.
مثل بات موسیقی دیسکورد، ولی داخل ماینکرفت!

## ✨ امکانات

- 🎶 پخش فایل‌های MP4/MP3/OGG/WAV در گروه‌های Voice Chat
- 👥 فقط اعضای گروه صدا رو می‌شنون
- 📋 صف آهنگ با قابلیت رد کردن، توقف و ادامه
- 🔊 تنظیم صدا برای هر بازیکن
- 📊 نمایش آهنگ در حال پخش با BossBar
- 🔒 امن — هیچ اطلاعات سیستمی به بازیکن نشون داده نمیشه
- 🔍 جستجوی هوشمند آهنگ
- ⌨️ تکمیل خودکار نام آهنگ‌ها با Tab

## 📋 پیش‌نیازها

| مورد | نسخه |
|---|---|
| سرور ماینکرفت | Spigot/Paper 1.20.2+ |
| جاوا | 21+ |
| Simple Voice Chat | 2.5.x |
| FFmpeg | هر نسخه جدید |

## 📥 نصب

### مرحله ۱: نصب وابستگی‌ها
1. [Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat) رو روی سرور نصب کنید
2. [FFmpeg](https://ffmpeg.org/download.html) رو نصب کنید:
3. فایل FFmpeg به صورت zip خواهد بود در مسیر FFmpeg/bin/FFmpeg.exe را کپی کنید و به مسیر سرور Plugins/MusicBot بریزید 
   - **ویندوز:** از ffmpeg.org دانلود و به PATH اضافه کنید
   - **لینوکس:** `sudo apt install ffmpeg`

### مرحله ۲: نصب پلاگین
1. فایل `MusicBot.jar` رو از [Releases](../../releases) دانلود کنید
2. توی پوشه `plugins/` بذارید
3. سرور رو ریستارت کنید

### مرحله ۳: اضافه کردن آهنگ
1. فایل‌های موسیقی رو توی `plugins/MusicBot/music/` بذارید
2. دستور `/music reload` رو بزنید

## 🎮 دستورات

| دستور | توضیح |
|---|---|
| `/music play <نام>` | پخش آهنگ |
| `/music list` | لیست آهنگ‌ها |
| `/music stop` | توقف |
| `/music pause` | مکث |
| `/music resume` | ادامه |
| `/music skip` | رد کردن |
| `/music volume <0-100>` | صدا |
| `/music queue` | صف |
| `/music reload` | بارگذاری مجدد (ادمین) |

## ⚠️ نکات مهم

- بازیکن **باید توی یه گروه Voice Chat** باشه
- فقط اعضای گروه صدا رو می‌شنون
- اگه DJ از گروه خارج بشه، موسیقی متوقف میشه

## 📦 تاریخچه تغییرات

### نسخه ۱.۱.۰ — آپدیت سشن گروهی

#### 🔧 رفع باگ
- **رفع شد:** صف آهنگ‌ها برای هر بازیکن جدا بود. الان همه اعضای گروه یک صف مشترک دارن.
- **رفع شد:** چند بازیکن در یک گروه می‌تونستن همزمان آهنگ‌های مختلف پخش کنن.
- **رفع شد:** هر بازیکنی می‌تونست موزیک رو متوقف/مکث کنه. الان فقط DJ می‌تونه.
- **رفع شد:** مسیرهای سیستمی به بازیکنان نشون داده می‌شد.
- **رفع شد:** کش صوتی وقتی فایل عوض می‌شد آپدیت نمی‌شد.
- **رفع شد:** بازیکنانی که به ویس چت وصل نبودن باعث خطا می‌شدن.

#### ✨ ویژگی‌های جدید
- **سشن گروهی:** هر گروه ویس چت فقط یک سشن موزیک مشترک داره.
- **سیستم DJ:** بازیکنی که موزیک رو شروع می‌کنه DJ میشه و فقط اون کنترل داره.
- **همه می‌تونن به صف اضافه کنن:** هر عضو گروه می‌تونه با `/music play` آهنگ به صف اضافه کنه.
- **پشتیبانی چندزبانه:** هر بازیکن می‌تونه زبان خودشو با `/music lang` تغییر بده.
- **زبان فارسی:** ترجمه کامل فارسی اضافه شد.
- **BossBar:** نمایش آهنگ در حال پخش با نوار پیشرفت — فقط برای اعضای گروه.
- **جستجوی هوشمند:** جستجوی فازی و تطبیق کلمات.
- **تکمیل خودکار:** نام آهنگ‌ها هنگام تایپ `/music play` تکمیل میشن.
